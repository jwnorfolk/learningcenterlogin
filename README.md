# Learning Center Login — Starter

This repository contains a minimal Node.js + Express scaffold that serves a static frontend.

Quick start

1. Install dependencies

```bash
cd /Users/Guest_Account/Documents/GitHub/learningcenterlogin
npm install
```

2. Start in development (auto-restarts on change)

```bash
npm run dev
```

3. Visit `http://localhost:3000`

Files of interest

- `server.js`: Express server and API route
- `public/index.html`: Demo frontend
- `public/styles.css`: Styles
